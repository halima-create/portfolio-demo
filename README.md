# portfolio-demo


## 📌 About
Personal portfolio website for a fresher Full-stack Developer showcasing projects in Web Development, IoT, and Machine Learning.


## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6)
- localStorage


## 📄 Pages
- **Home** - Introduction & skills
- **Projects** - 8 projects across 3 domains
- **Todo App** - Interactive todo with localStorage
- **Product Filter** - Filter/sort products by price & category  
- **Contact** - Get in touch form


## ✨ Features
- Fully responsive design
- Active navigation highlighting
- Todo tasks saved in browser
- Real-time product filtering
- Contact form (demo)
